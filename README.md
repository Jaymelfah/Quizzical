# Quizzical Game
In this project I built a quiz game with React on Geography. This app grades answers and highlights correctly 
selected answers.

 ## Learning objectives
- Use React components.
- Use React props.
- Use React state.
- Use Events in a React App
- Use React Hooks

## Built With

- Create React App
- JSX syntax

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Have a Desktop or Laptop Computer running on with Windows, Mac OS or Linux operating system.
- Have git installed on the Computer
- Have a browser installed on the Computer

### Setup

- If it suits you, on your local computer, create a directory in your preferred location (e.g in the Desktop or Document directory)

### Install
- To clone the project on your machine: Type `git clone https://github.com/Jaymelfah/Math-Magician.git`.
- Run `npm install` in your terminal
- To run the project on  live server:
`npm run start`
